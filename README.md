# ANARI Platform

ANARI is a personal life management app (Version 4) with:

- Dashboard
- Tasks, Bills, Subscriptions
- Vehicle Maintenance Predictor
- AI Assistant
- Push Notifications
- Calendar Timeline
- Web + iOS + Android

## Setup

1. Install Firebase CLI: `npm install -g firebase-tools`
2. Clone repository: `git clone https://github.com/yourusername/anari-platform.git`
3. Enter folder: `cd anari-platform`
4. Install functions dependencies: `cd functions && npm install`
5. Deploy: `firebase deploy`
